# Katya Business Bot

Telegram-бот-витрина с приветствием, фото, кнопками, вопросами админу, базой пользователей и рассылкой.

## Файлы

- `main.py` — основной код бота
- `requirements.txt` — зависимости
- `.env.example` — пример переменных
- `Procfile` — запуск на Railway

## Переменные для Railway

Создай в Railway → Variables:

```env
BOT_TOKEN=токен_бота_из_BotFather
ADMIN_ID=779528794
START_PHOTO=
SUPPORT_USERNAME=
PROTECT_CONTENT=true
DB_PATH=katya_bot.db
```

### START_PHOTO

Можно оставить пустым — тогда бот будет отправлять только текст.

Можно указать:

1. ссылку на фото;
2. Telegram file_id;
3. путь к файлу, например `start.jpg`, если фото лежит рядом с `main.py`.

## Команды

Для пользователя:

- `/start` — запуск бота
- `/menu` — открыть меню

Для админа:

- `/users` — количество пользователей
- `/reply USER_ID текст` — ответить пользователю
- `/broadcast текст` — рассылка по базе

## Защита от копирования

В коде включено:

```env
PROTECT_CONTENT=true
```

Это запрещает пересылку и сохранение сообщений бота там, где это поддерживается Telegram.  
Скриншоты полностью запретить нельзя, но в закрытом канале можно включить настройку Telegram “Запретить копирование”.

PDF-файлы лежат локально внутри проекта в папке `documents/`.

Привязка:

- 1 кнопка: `documents/cosmetics.pdf`
- 2 кнопка: `documents/russia_suppliers.pdf`
- 3 кнопка: `documents/europe_factories.pdf`

Внутри каждой из этих трёх веток есть кнопка `📄 Открыть PDF`.

## Выдача PDF только после оплаты

В 3 ветках PDF закрыты до подтверждения оплаты:

- База поставщиков оригинальной косметики
- База поставщиков брендов РФ
- Фабрики Европы

Как работает:

1. Пользователь нажимает `💳 Оплатить и получить PDF`.
2. Бот показывает тариф, карту ВТБ и сумму.
3. Пользователь отправляет чек в бот.
4. Админу `779528794` приходит чек и кнопки:
   - `✅ Подтвердить оплату`
   - `❌ Отклонить`
5. После подтверждения бот автоматически отправляет пользователю нужный PDF.
6. После подтверждения этот пользователь сможет открыть PDF повторно через кнопку `📄 Открыть PDF`.

Ручная выдача доступа админом:

```text
/grant USER_ID cosmetics_suppliers
/grant USER_ID brand_suppliers_rf
/grant USER_ID factories_europe
```
